# EEGR4613 - Software Defined Radio
Hello! This repository contains lab work for EEGR4613 - Communications Engineering, pertaining to Software Defined Radio (SDR) concepts using GNU Radio and the HackRF platform.

## Overview
The purpose of this repository is to document and demonstrate SDR concepts using GNU Radio and the Hack RF platform to demonstrate fundamental concepts of communications engineering.

## Repository Structure
The repository is structured in the following manner:

Lab 01/

├── DryRun/

  │   ├── dryRun.grc

  │   └── README.md

├── RunDry/

  │   ├── runDry.grc

  │   └── README.md

Lab 02/

├── Individual Folder for Each Flowgraph/

  │   ├── '.grc' file

  │   ├── '.py' file (if applicable)

  │   ├── '.png' file(s) of results (if applicable)

  │   └── README.md

## Usage
1.) Use git clone to clone the repo to your computer

2.) Navigate to a specific lab folder (e.g., Lab02)

3.) Open a flowgraph folder

4.) Read the included README.md for details

5.) Use GNU Radio Companion to run the '.grc' file

6.) Modify flowgraph or compare output to examples
